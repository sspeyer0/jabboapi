L'API de Jabbo vous permettent d'écrire et de lire les données de la base de donnée de https://jabbo.eu/. Afficher les articles, vérifier les informations de connexion, retrouver un joueur etc ...

Pour utiliser certaine fonction de cette API, vous aurais besoins d'une clé d'identification secrète. Seul les développeur de https://jabbo.eu/ possèdents une clef secrète. Tous partage d'une clé d'identification secrète sera sanctionné.

/getAvatarByUsername

Pour récuperer un avatar avec un nom d'utilisateur, faire une requête POST sur l'adresse suivante:
https://api.jabbo.eu/getAvatarByUsername 


              Exemple d'utilisation:

              Requête POST avec les données suivantes sur le lien 
 https://api.jabbo.eu/getAvatarByUsername:
              
              {
                username: "Sple"
              }
              

              Résultat de la requête:
              
              
              {
                avatar: "fa-1212-62.hr-3163-37.ca-1813-62.hd-190-1.wa-2001-62.ch-255-br100.lg-3057-81.he-1607-62.ha-3488-62.ea-6136543-62"
              }

PARAMETRES

Champ Type Description
username String Nom d'utilisateur du joueur

Erreurs

              Exemple:

              Requête POST avec aucunes données sur le lien 
 https://api.jabbo.eu/getAvatarByUsername:

              Résultat de la requête:
              
              {
                error: "ERR000"
              }

L'API de https://jabbo.eu/ utilise un système de code erreur pour aider les développeurs dans son utilisation.

Code erreur Signification
ERR000 Certains paramètres sont manquants.
ERR001 Clé secrète invalide.
